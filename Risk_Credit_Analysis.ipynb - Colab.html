Commencons par identifier le type de variable : Nominale ou Ordinale


    

  • La variable dépendante Loan Status est une variables nominale car il ya pas d'ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie YES n'est ni supérieur ni inférieur à NO
    

    • 

  • La variable GENDER est une variables nominale car il ya pas d'ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie FEMME n'est ni supérieur ni inférieur à HOMME
    

    • 

  • La variable MARRIED est une variables nominale car il ya pas d'ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie YES n'est ni supérieur ni inférieur à NO
    

    • 

  • La variable DEPENDENTS est une variables ordinale car il ya un ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie Avoir 01 enfants implique implicitement une étape à franchier pour en avoir 02 ou plus
    

    • 

  • La variable EDUCATION est une variables ordinale car il ya un ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie être GRADUATE implique implicitement une étape à franchier pour les personnes NON GRADUATE
    

    • 

  • La variable SELF EMPLOYEE est une variables nominale car il ya pas d'ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie YES n'est ni supérieur ni inférieur à NO
    

    • 

  • La variable RPOPERTY AREA est une variables ordinale car il ya un ordre logique entre les valeurs attribuées à cette varaiable dans notre contexte ie être dans un environnement URBAN implique implicitement une étape à franchier pour les personnes SEMI-URBAN
    

    •

    • 

  • La variable GENDER est une variables nominale car il ya pas d'ordre logique entre les valeurs attribues  cette varaiable dans notre contexte ie FEMME n'est ni suprieur ni infrieur  HOMME
    

    • 

  • La variable MARRIED est une variables nominale car il ya pas d'ordre logique entre les valeurs attribues  cette varaiable dans notre contexte ie YES n'est ni suprieur ni infrieur  NO
    

    • 

  • La variable DEPENDENTS est une variables ordinale car il ya un ordre logique entre les valeurs attribues  cette varaiable dans notre contexte ie Avoir 01 enfants implique implicitement une tape  franchier pour en avoir 02 ou plus
    

    • 

  • La variable EDUCATION est une variables ordinale car il ya un ordre logique entre les valeurs attribues  cette varaiable dans notre contexte ie tre GRADUATE implique implicitement une tape  franchier pour les personnes NON GRADUATE
